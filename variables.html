In PHP, variables are assigned values with the assignment operator (=). Variable names can contain numbers, letters, and underscores (_). A sigil ($) must always precede a variable name. They cannot start with a number and they cannot have spaces or any special characters. The convention in PHP is to use snake case for variable naming; this means that lowercase words are delimited with an underscore character (_). Variable names are case-sensitive.
